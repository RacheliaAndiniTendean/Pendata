  1. Menggunakan library LinearRegression dari sklearn

  2. Menghitung secara analitik menggunakan rumus matematika dan matriks

3. Regresi Linear#

Regresi linear sederhana digunakan untuk mencari hubungan antara variabel X dan variabel Y.

Persamaan umum regresi linear:

\[ y = \beta_0 + \beta_1x \]

Keterangan:

  • (y) = variabel output / hasil prediksi

  • (x) = variabel input

  • (\beta_0) = intercept / konstanta

  • (\beta_1) = koefisien regresi / slope

4. Perhitungan Menggunakan Rumus#

Perhitungan manual digunakan untuk mencari nilai koefisien regresi tanpa bantuan library.

Penjelasan:

Kolom (x^2):

\[ x^2 = x \times x \]

Contoh:

\[ 2^2 = 2 \times 2 = 4 \]

Kolom (xy):

\[ xy = x \times y \]

Contoh:

\[ 2 \times 2 = 4 \]

Diperoleh:

\[ \sum x = 24 \]
\[ \sum y = 28 \]
\[ \sum x^2 = 88 \]
\[ \sum xy = 102 \]

4.2 Menghitung Beta 1#

Beta 1 digunakan untuk mengetahui perubahan nilai (y) ketika nilai (x) bertambah 1.

Rumus:

\[ \beta_1 = \frac{n\sum xy - (\sum x)(\sum y)} {n\sum x^2 - (\sum x)^2} \]

Substitusi:

\[ \beta_1 = \frac{7(102) - (24)(28)} {7(88) - (24)^2} \]

Hitung pembilang:

\[ 7(102) = 714 \]
\[ 24(28) = 672 \]
\[ 714 - 672 = 42 \]

Hitung penyebut:

\[ 7(88) = 616 \]
\[ 24^2 = 576 \]
\[ 616 - 576 = 40 \]

Maka:

\[ \beta_1 = \frac{42}{40} \]
\[ \beta_1 = 1.05 \]

4.3 Menghitung Beta 0#

Beta 0 disebut intercept atau konstanta.

Rumus:

\[ \beta_0 = \bar{y} - \beta_1\bar{x} \]

Menghitung rata-rata x:

\[ \bar{x} = \frac{24}{7} \]
\[ \bar{x} = 3.4286 \]

Menghitung rata-rata y:

\[ \bar{y} = \frac{28}{7} \]
\[ \bar{y} = 4 \]

Substitusi:

\[ \beta_0 = 4 - (1.05)(3.4286) \]
\[ = 4 - 3.6 \]
\[ \beta_0 = 0.4 \]

4.4 Persamaan Regresi Linear#

Setelah diperoleh nilai:

\[ \beta_0 = 0.4 \]
\[ \beta_1 = 1.05 \]

Maka persamaan regresi linear:

\[ y = 0.4 + 1.05x \]

4.5 Interpretasi Hasil#

Dari hasil perhitungan diperoleh:

  • Nilai intercept sebesar 0.4

  • Nilai koefisien regresi sebesar 1.05

Artinya:

  • Ketika x = 0, maka nilai prediksi y = 0.4

  • Setiap kenaikan 1 nilai x akan meningkatkan nilai y sebesar 1.05

Karena koefisien regresi bernilai positif, maka hubungan x dan y bersifat positif atau searah.

5. Cara Menggunakan GeoGebra#

GeoGebra digunakan untuk menampilkan titik data dalam bentuk scatter plot dan membuat garis regresi linear.

5.1 Memasukkan Titik Data#

Masukkan titik data satu per satu pada kolom input GeoGebra.

A = (2,2)
B = (4,3)
C = (3,5)
D = (3,4)
E = (3,3)
F = (4,5)
G = (5,6)

Setelah semua titik dimasukkan, titik-titik tersebut akan muncul pada bidang koordinat.

5.2 Membuat List Data#

Agar semua titik dapat digunakan dalam proses regresi, titik-titik tersebut dibuat menjadi satu list.

L = {A,B,C,D,E,F,G}

List (L) berisi seluruh titik data yang akan digunakan untuk membuat garis regresi.

5.3 Membuat Garis Regresi Linear#

Untuk membuat garis regresi linear, masukkan perintah berikut:

FitLine(L)

atau jika menggunakan GeoGebra bahasa Indonesia:

CocokGarisY(L)

GeoGebra akan menampilkan garis regresi linear berdasarkan titik-titik yang sudah dimasukkan.

5.4 Hasil GeoGebra#

Dari GeoGebra diperoleh persamaan regresi:

\[ y = 0.4 + 1.05x \]

Hasil ini sama dengan perhitungan manual menggunakan rumus, yaitu:

\[ \beta_0 = 0.4 \]
\[ \beta_1 = 1.05 \]

Artinya, GeoGebra menghasilkan garis regresi yang sesuai dengan hasil perhitungan manual.

6. Perhitungan Menggunakan sklearn Python#

import numpy as np
from sklearn.linear_model import LinearRegression
import matplotlib.pyplot as plt

# Data X dan Y
X = np.array([2, 4, 3, 3, 3, 4, 5]).reshape(-1, 1)
y = np.array([2, 3, 5, 4, 3, 5, 6])

# Membuat model regresi linear
model = LinearRegression()

# Training / fitting model
model.fit(X, y)

# Mengambil nilai beta
beta0 = model.intercept_
beta1 = model.coef_[0]

# Menampilkan hasil
print("Intercept / Beta 0 =", beta0)
print("Koefisien Regresi / Beta 1 =", beta1)

# Persamaan regresi
print(f"Persamaan regresi: y = {beta0:.2f} + {beta1:.2f}x")

# Prediksi garis regresi
y_pred = model.predict(X)

# Visualisasi
plt.scatter(X, y, color='blue', label='Data Asli')
plt.plot(X, y_pred, color='red', label='Garis Regresi')

plt.xlabel('X')
plt.ylabel('Y')
plt.title('Linear Regression')
plt.legend()

plt.show()

Intercept / Beta 0 = 0.3999999999999999
Koefisien Regresi / Beta 1 = 1.05
Persamaan regresi: y = 0.40 + 1.05x

Penjelasan Kode#

Library yang digunakan:

import numpy as np
from sklearn.linear_model import LinearRegression
import matplotlib.pyplot as plt

Keterangan:

  • numpy digunakan untuk membuat dan mengolah array data.

  • LinearRegression digunakan untuk membuat model regresi linear.

  • matplotlib.pyplot digunakan untuk membuat visualisasi grafik.

Memasukkan dataset:

X = np.array([2, 4, 3, 3, 3, 4, 5]).reshape(-1, 1)
y = np.array([2, 3, 5, 4, 3, 5, 6])

Keterangan:

  • X digunakan sebagai variabel input.

  • y digunakan sebagai variabel output.

  • .reshape(-1, 1) digunakan agar data berbentuk matriks kolom sehingga dapat diproses oleh sklearn.

Membuat model regresi linear:

model = LinearRegression()

Kode tersebut digunakan untuk membuat objek model regresi linear.

Melakukan training / fitting model:

model.fit(X, y)

Fungsi fit() digunakan untuk melatih model berdasarkan dataset.

Pada tahap ini model akan mencari nilai:

\[ \beta_0 \]

dan

\[ \beta_1 \]

agar diperoleh garis regresi terbaik dengan error sekecil mungkin.

Mengambil nilai beta:

beta0 = model.intercept_
beta1 = model.coef_[0]

Keterangan:

  • model.intercept_ digunakan untuk mengambil nilai intercept atau (\beta_0)

  • model.coef_[0] digunakan untuk mengambil nilai koefisien regresi atau (\beta_1)

Menampilkan hasil regresi:

print("Intercept / Beta 0 =", beta0)
print("Koefisien Regresi / Beta 1 =", beta1)

Output:

Koefisien Regresi / Beta 1 = 1.05

Menampilkan persamaan regresi:

print(f"Persamaan regresi: y = {beta0:.2f} + {beta1:.2f}x")

\[ y = 0.4 + 1.05x \]

Melakukan prediksi:

y_pred = model.predict(X)

Fungsi predict() digunakan untuk menghasilkan nilai prediksi berdasarkan model regresi linear yang sudah dilatih.

Membuat scatter plot:

plt.scatter(X, y, color='blue', label='Data Asli')

Kode tersebut digunakan untuk menampilkan titik data asli pada grafik.

Membuat garis regresi:

Kode tersebut digunakan untuk membuat garis regresi linear berdasarkan hasil prediksi model.

Memberikan label grafik:

7. Kesimpulan#

Berdasarkan hasil perhitungan menggunakan rumus manual, GeoGebra, dan sklearn Python diperoleh hasil yang sama yaitu:

\[ \beta_0 = 0.4 \]
\[ \beta_1 = 1.05 \]

Sehingga persamaan regresi linear yang diperoleh adalah:

\[ y = 0.4 + 1.05x \]

Hasil tersebut menunjukkan bahwa variabel x memiliki hubungan positif terhadap variabel y.

Nilai koefisien regresi beta_1 sebesar 1.05 menunjukkan bahwa setiap kenaikan 1 nilai x, maka nilai y diprediksi meningkat sebesar 1.05.

Sedangkan nilai intercept beta_0 sebesar 0.4 menunjukkan bahwa ketika nilai x = 0, maka nilai prediksi y adalah 0.4.

Dari hasil visualisasi GeoGebra dan grafik Python juga terlihat bahwa garis regresi mengikuti pola penyebaran data, sehingga model regresi linear dapat digunakan untuk melakukan prediksi terhadap data tersebut.

Dengan demikian dapat disimpulkan bahwa metode regresi linear berhasil digunakan untuk mencari hubungan antara variabel x dan variabel y, baik menggunakan perhitungan manual maupun menggunakan library sklearn Python.
